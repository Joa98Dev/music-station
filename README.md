# Music Station 📻

This is a simple music player made with python, using the Tkinter and Pygame libraries.

## How to use it? 🤔

Turn the virtual environment on:

* **Linux** and **Mac**: `source music_player/bin/activate`

* **Windows**: `music_player\Scripts\activate`

1. Press the **load** button.

![Screenshot](screenshots/shot_01.png)

2. Look for your music folder and select it.

![Screenshot](screenshots/shot_02.png)

3. Done! Now your playlist has been loaded.

![Screenshot](screenshots/shot_04.png)

You can Select which song of the playlist you want to listen to, as well as filter the format of the audio.

Also you can reset the music player to load another playlist, pause, resume, stop the music/playlist.

The Music player can play:

* mp3

* ogg

* wav
