# Music Station 📻
This is a simple but pretty music player.
Made it with Python and using the Tkinter and Pygame libraries.

## How to use it? 🤔
1. Press the **load** button.
![Screenshot01](screenshots/image.png)
2. Look for your music folder and select it.
![Screenshot02](screenshots/image2.png)
3. Done! Now your playlist has been loaded.
![Screenshot03](screenshots/image3.png)

You can Select which song of the playlist you want to listen to.

Also you can reset the playlist to load another, puase, resume or stop the songs.

The Music player can play:
* mp3
* ogg
* wav
